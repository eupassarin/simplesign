Estrutura do Projeto

Exemplo de listas ordenadas e não-ordenadas com vários níveis de aninhamento.

Tecnologias Utilizadas

  • Backend
    • .NET 10
      • ASP.NET Core
      • Entity Framework Core
      • SignalR
    • PostgreSQL
    • Redis
  • Frontend
    • React 19
    • TypeScript
    • Tailwind CSS
  • Infraestrutura
    • Docker
    • Kubernetes
    • Azure

Passos de Instalação

  1. Clonar o repositório
  2. Instalar dependências
    1. Executar dotnet restore
    2. Executar npm install no diretório frontend
    3. Configurar variáveis de ambiente
      1. Copiar .env.example para .env
      2. Preencher as credenciais do banco
      3. Configurar chave JWT
  3. Executar migrations do banco de dados
  4. Iniciar o servidor
  5. Acessar http://localhost:5000

Requisitos

  • .NET SDK 10.0 ou superior
  • Node.js 22+
  • Docker Desktop
  • PostgreSQL 16+
